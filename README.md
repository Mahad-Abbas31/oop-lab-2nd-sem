# OOP Lab 2nd Semester – C++ Practice Repository

This repository contains my Object Oriented Programming (OOP) lab work for the 2nd semester.  
All programs are written in **C++** and cover fundamental to intermediate OOP concepts including constructors, dynamic memory, file handling, classes, objects, and more.

## 📚 Repository Description

This repo includes:

- Lab tasks
- Practice programs
- Quiz programs
- Constructor & Copy Constructor examples
- Dynamic memory programs
- File handling programs
- Object oriented design practice

Each file is named according to:

RollNo_Name_LabNo_QuestionNo.cpp

Example:
2501366_MahadAbbas_L6_Q1.cpp

Meaning:
- 2501366 → Roll number
- L6 → Lab 6
- Q1 → Question 1
